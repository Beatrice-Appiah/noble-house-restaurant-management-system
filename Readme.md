# Noble House Restaurant Management System

## Project Overview
This project is a web-based Restaurant Management System developed as part of the Level 300 Field Trip & Report Writing course at GCTU.

## Technologies Used
- Node.js
- Express.js
- MySQL
- HTML, CSS, JavaScript

## System Features
- User authentication
- Customer order management
- Inventory management
- Dashboard navigation

## How to Run the System
1. Install Node.js
2. Install MySQL and create the database
3. Import SQL tables
4. Run:

   ```bash
   node server.js
   ```

Open browser and visit:
  - http://localhost:3000

Default Login:
  - Username: admin
  - Password: admin123
